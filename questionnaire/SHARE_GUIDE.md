# 问卷分享功能使用指南

## 📢 功能概述

系统已完善问卷分享功能，提供多种方式帮助您分享和收集问卷！

## 🎯 如何获取访问码

### 方式一：在问卷列表中查看

1. 登录系统
2. 进入"问卷管理"页面
3. 在列表的"访问码"列可以看到每个问卷的8位访问码
4. 点击"复制访问码"按钮可快速复制

### 方式二：查看问卷详情

1. 在问卷列表中，点击已发布问卷的"🔗 分享"按钮
2. 进入分享页面，可以看到：
   - **大号显示的访问码**
   - 访问链接
   - 二维码（演示）
   - 完整的分享说明

### 方式三：创建问卷后直接查看

1. 创建新问卷并保存
2. 系统会询问"是否立即查看分享信息？"
3. 点击"确定"即可跳转到分享页面

## 📋 三种分享方式

### 1️⃣ 分享访问码

**适用场景**：微信群、QQ群、短信等

**操作步骤**：
1. 复制访问码（如：`abc12345`）
2. 告诉用户访问系统首页
3. 在"访问问卷"区域输入访问码
4. 点击"开始填写"

**示例消息**：
```
【问卷调查】
请大家填写满意度调查问卷
访问码：abc12345
地址：http://localhost:5173
```

### 2️⃣ 分享访问链接

**适用场景**：邮件、即时通讯工具、网页

**操作步骤**：
1. 点击"复制链接"按钮
2. 直接发送链接给用户
3. 用户点击链接即可直接打开问卷

**示例链接**：
```
http://localhost:5173/questionnaire/fill/abc12345
```

**优点**：
- 用户无需手动输入访问码
- 一键直达问卷
- 更方便快捷

### 3️⃣ 分享二维码

**适用场景**：海报、传单、PPT演示

**操作步骤**：
1. 在分享页面查看二维码
2. 截图或保存二维码
3. 将二维码添加到宣传材料中
4. 用户扫码即可填写

**注意**：当前版本的二维码为演示版，建议使用草料二维码等工具生成正式的二维码。

## 💡 实用技巧

### 技巧1：批量分享

如果需要同时分享多个问卷：
1. 在问卷列表中逐个复制访问码
2. 整理成清单发送给用户

示例：
```
【本周调查问卷】
1. 满意度调查 - 访问码：aaa11111
2. 需求调查 - 访问码：bbb22222
3. 反馈调查 - 访问码：ccc33333
```

### 技巧2：快速预览

在发布前，点击"预览"按钮可以：
- 查看问卷实际效果
- 测试填写流程
- 检查题目显示

### 技巧3：权限控制

- **公开问卷**：任何人都可以通过访问码填写
- **已发布状态**：问卷必须发布后才能填写
- **已结束状态**：问卷结束后无法继续填写

### 技巧4：追踪填写情况

1. 点击"统计"按钮
2. 查看"总参与人数"
3. 实时监控收集进度

## ⚠️ 注意事项

### 发布前必读

1. **草稿状态无法分享**
   - 问卷创建后默认为"草稿"状态
   - 必须点击"发布"后才能分享
   - 发布后访问码才有效

2. **访问码说明**
   - 每个问卷有唯一的8位访问码
   - 创建时自动生成
   - 不会重复

3. **问卷状态**
   - 草稿：仅创建者可见，无法填写
   - 已发布：可以正常填写
   - 已结束：无法继续填写

### 常见问题

**Q1: 访问码在哪里？**
A: 在问卷列表的"访问码"列，或点击"🔗 分享"查看。

**Q2: 如何修改访问码？**
A: 访问码自动生成，无法修改。每个问卷的访问码唯一且固定。

**Q3: 问卷必须登录才能填写吗？**
A: 不需要。填写问卷无需登录，只需要访问码即可。

**Q4: 如何知道有多少人填写了？**
A: 点击"统计"按钮，查看"总参与人数"。

**Q5: 可以重复填写吗？**
A: 系统会基于IP地址检测，防止同一用户重复提交。

**Q6: 链接失效了怎么办？**
A: 检查问卷状态是否为"已发布"。如果是"已结束"，需要重新发布。

**Q7: 二维码扫不了？**
A: 当前二维码为演示版，建议使用第三方工具（如草料二维码）生成正式二维码。
